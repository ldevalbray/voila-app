# Polices CommitMono et Fustat

## CommitMono
- `CommitMono-400-Regular.otf` (déjà présent)

La police CommitMono est disponible gratuitement sous licence OFL depuis :
- Fontesk : https://fontesk.com/commit-mono-font/
- GitHub : https://github.com/eendroroy/commit-mono

## Fustat
